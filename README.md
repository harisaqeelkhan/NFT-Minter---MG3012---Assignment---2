## Overview
NFT Minter is a decentralized application (dApp) that allows users to **mint NFTs directly on the Ethereum blockchain**. Users can connect their **MetaMask wallet**, upload images, and mint their own NFTs on the **Sepolia test network**.  

This project is built with **HTML, CSS, JavaScript, and Solidity** using the ERC-721 standard.

---
## Repository File Structure
- NFTContract.sol - the smartcontract code
- index.html - the website interface  
- README.md - this file
- Word File - pictures showing it works
- style.css - showing how the site is styled

---
## Features
- **Connect MetaMask Wallet**: Seamlessly connect your Ethereum wallet.
- **Upload Assets**: Upload images to mint as NFTs.
- **Mint NFTs**: Mint NFTs to your connected wallet.
- **Transaction Feedback**: Real-time status updates (pending, success, failed).
- **Gallery Display**: View minted NFTs in a card gallery.
